# GitHub 专家指南

从入门到精通的 GitHub 使用教程，涵盖基础操作、仓库管理、工作流、协作开发等核心内容。

## 文档

在线文档（Read the Docs）：

> **https://github-expert.readthedocs.io/zh-cn/latest/**

## 目录结构

```
source/
├── preface/                   # 前言
├── chapter_01_basics/         # GitHub 基础入门
├── chapter_02_repo/           # 仓库管理
├── chapter_03_workflow/       # 工作流
├── chapter_04_collaboration/ # 协作开发
├── chapter_05_advanced/      # 高级技巧
├── examples/                 # 可运行示例代码
└── appendix/                  # 附录
```

## 本地构建

```bash
pip install -r requirements.txt
make html       # 构建 HTML
make serve      # 构建并在 localhost:8000 启动预览
```

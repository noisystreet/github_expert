术语表
======

.. glossary::

   Repository
      仓库，存储项目代码和版本历史的地方

   Branch
      分支，独立的开发线

   Commit
      提交，保存代码变更的快照

   Pull Request
      拉取请求，请求合并代码的机制

   Merge
      合并，将一个分支的变更合并到另一个分支

   Clone
      克隆，复制远程仓库到本地

   Fork
      复制别人的仓库到自己的账户

   Issue
      问题，跟踪任务、Bug 和讨论

   CI/CD
      持续集成/持续部署

   Actions
      GitHub 的自动化工作流功能

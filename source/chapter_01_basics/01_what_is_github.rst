什么是 GitHub
=============

GitHub 是一个基于 Git 的代码托管平台，提供版本控制和协作功能。

核心特性
--------

- **版本控制**：基于 Git 的分布式版本控制
- **协作开发**：支持多人协作开发
- **代码审查**：Pull Request 和 Code Review
- **项目管理**：Issues、Projects、Wiki

GitHub vs Git
-------------

Git 是版本控制系统，而 GitHub 是基于 Git 的托管平台。

.. code-block:: bash

   # Git 是本地工具
   git init
   git commit -m "first commit"

   # GitHub 提供远程托管
   git remote add origin https://github.com/user/repo.git
   git push -u origin main
